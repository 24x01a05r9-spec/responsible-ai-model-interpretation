# Responsible AI & Model Interpretation

## Overview
This project demonstrates Responsible AI techniques using model explainability and bias analysis. LIME was used to explain predictions and fairness was checked across sensitive groups.

## Technologies Used
- Python
- Scikit-learn
- LIME
- NumPy
- Pandas
- Google Colab

## Features
- Model explainability using LIME
- Bias analysis across Male and Female groups
- Fairness evaluation
- Mitigation recommendations

## Important Features
- Age
- Gender
- Income

## Results
- Class 0 Probability: 0.37
- Class 1 Probability: 0.63

## Bias Analysis
The model showed performance differences between Male and Female groups, indicating possible bias.

## Mitigation Techniques
- Balance dataset
- Add diverse data
- Remove biased variables
- Perform fairness testing
